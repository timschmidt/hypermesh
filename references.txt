Hypermesh exact mesh kernel references
======================================

Mesh Boolean And Arrangement Kernels
------------------------------------

1. Trettner, P.; Nehring-Wirxel, J.; Kobbelt, L. "EMBER: Exact Mesh Booleans via Efficient & Robust Local Arrangements." ACM Transactions on Graphics 41(4), SIGGRAPH 2022. doi:10.1145/3528223.3530181.
   Primary performance target for adaptive exact local arrangements, early-outs, exact plane-based representation, and parallel work distribution.
   https://www.graphics.rwth-aachen.de/publication/03339/

2. Cherchi, G.; Pellacini, F.; Attene, M.; Livesu, M. "Interactive and Robust Mesh Booleans." ACM Transactions on Graphics 41(6), 2022. doi:10.1145/3550454.3555460.
   Practical performance reference for robust exact-predicate boolean pipelines.

3. Zhou, Q.; Grinspun, E.; Zorin, D.; Jacobson, A. "Mesh Arrangements for Solid Geometry." ACM Transactions on Graphics 35(4), SIGGRAPH 2016. doi:10.1145/2897824.2925901.
   Robust arrangement and winding-vector correctness model for arbitrary mesh inputs.
   https://www.cs.columbia.edu/cg/mesh-arrangements/

4. Levy, B. "Exact Predicates, Exact Constructions and Combinatorics for Mesh CSG." arXiv:2405.12949, 2024.
   Exact co-refinement, Weiler arrangement construction, radial sorting around non-manifold intersection edges, and CDT remeshing.
   https://arxiv.org/abs/2405.12949

5. Barki, H.; Guennebaud, G.; Foufou, S. "Exact, Robust, and Efficient Regularized Booleans on General 3D Meshes." Computers & Mathematics with Applications 70(6), 2015. doi:10.1016/j.camwa.2015.06.016.
   Reference for regularized boolean semantics on degenerate closed and orientable meshes.

6. Bernstein, G.; Fussell, D. "Fast, Exact, Linear Booleans." Computer Graphics Forum 28(5), 2009. doi:10.1111/j.1467-8659.2009.01504.x.
   BSP-style exact boolean reference with a compact predicate substrate.

7. Nehring-Wirxel, J.; Trettner, P.; Kobbelt, L. "Fast Exact Booleans for Iterated CSG using Octree-Embedded BSPs." Computer-Aided Design 135, 2021. doi:10.1016/j.cad.2021.103015.
   Specialized backend direction for repeated CSG and plane-cut workloads.

8. Hachenberger, P.; Kettner, L.; Mehlhorn, K. "Boolean Operations on 3D Selective Nef Complexes." Computational Geometry 38(1-2), 2007. doi:10.1016/j.comgeo.2006.11.009.
   Semantics reference for non-manifold, mixed-dimensional, open/closed, and regularized topology.

Predicate And Construction Substrate
------------------------------------

9. Attene, M. "Indirect Predicates for Geometric Constructions." Computer-Aided Design 126, 2020. doi:10.1016/j.cad.2020.102856.
   Compare constructed intersection points through retained source primitives instead of eagerly materializing every exact coordinate.
   https://arxiv.org/abs/2105.09772

10. Shewchuk, J. R. "Adaptive Precision Floating-Point Arithmetic and Fast Robust Geometric Predicates." Discrete & Computational Geometry 18(3), 1997. doi:10.1007/PL00009321.
    Foundational adaptive exact predicate work.
    https://www.cs.cmu.edu/~quake/robust.html

11. Yap, C. K. "Towards Exact Geometric Computation." Computational Geometry 7(1-2), 1997. doi:10.1016/0925-7721(95)00040-2.
    Foundational exact geometric computation framing.

12. Guigue, P.; Devillers, O. "Fast and Robust Triangle-Triangle Overlap Test Using Orientation Predicates." Journal of Graphics Tools 8(1), 2003. doi:10.1080/10867651.2003.10487580.
    Narrow-phase triangle overlap staging via orientation predicates.

13. Moller, T. "A Fast Triangle-Triangle Intersection Test." Journal of Graphics Tools 2(2), 1997. doi:10.1080/10867651.1997.10487472.
    Classic fast triangle-triangle rejection baseline.

Broad Phase And Spatial Indexing
--------------------------------

14. Klosowski, J. T.; Held, M.; Mitchell, J. S. B.; Sowizral, H.; Zikan, K. "Efficient Collision Detection Using Bounding Volume Hierarchies of k-DOPs." IEEE Transactions on Visualization and Computer Graphics 4(1), 1998. doi:10.1109/2945.675649.
    BVH and k-DOP direction for tighter exact broad-phase rejection.

15. Xu, S.; Keyser, J. "Fast and Robust Booleans on Polyhedra." Computer-Aided Design 45(2), 2013. doi:10.1016/j.cad.2012.10.036.
    Spatial hashing and exact-topology separation reference.

Planar Arrangement And Triangulation Support
--------------------------------------------

16. Held, M. "FIST: Fast Industrial-Strength Triangulation of Polygons." Algorithmica 30(4), 2001. doi:10.1007/s00453-001-0028-4.
    Practical polygon triangulation behavior.

17. Boissonnat, J.-D.; Devillers, O.; Pion, S.; Teillaud, M.; Yvinec, M. "Triangulations in CGAL." Computational Geometry 22(1-3), 2002. doi:10.1016/S0925-7721(01)00054-2.
    Robust triangulation package design reference.

18. Lee, D. T.; Lin, A. K. "Generalized Delaunay Triangulation for Planar Graphs." Discrete & Computational Geometry 1, 1986.
    Constrained/generalized Delaunay topology for face-cell remeshing.

Practical Baselines
-------------------

19. CGAL Polygon Mesh Processing: Corefinement and Boolean Operations.
    Corefinement baseline for refining meshes so intersection polylines become edges before set extraction.
    https://doc.cgal.org/latest/PMP_Boolean_operations/group__PMP__corefinement__grp.html

20. CGAL 3D Nef Polyhedra.
    Baseline for closure under boolean, complement, boundary, closure, interior, and regularization operations.
    https://doc.cgal.org/latest/Nef_3/index.html

21. Manifold Library.
    Practical robust manifold-output boolean implementation for engineering comparison.
    https://github.com/elalish/manifold

22. Solidean.
    Production signal for exact boolean engineering choices descended from EMBER-style kernels.
    https://solidean.com/
